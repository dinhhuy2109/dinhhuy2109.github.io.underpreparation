Getting Started

This document describes the details that the user must know in order to use this software. The main objective is to support participants in the competition Cybertech held annually between groups from different universities. This competition is organized by the Division of Systems and Automation Engineering and the UPM Reset student association, specifically in the field of programming and control, so that the user can do the necessary tests to the code (only) using an environment of easy access.

Source Code

The Subversion repository for YellowSimulator is hosted on assembla. In this repository you can be find the latest version of YellowSimulator.

If you have subversion installed, you should do the following to get the initial copy of the repository:

# Non-members may check out a read-only working copy anonymously over HTTP.
svn checkout http://subversion.assembla.com/svn/vct-yellow-team/ vct-yellow-team-read-only

Public access to the SVN repository is read-only at the moment. So it is not possible to commit changes, but you can send us patches (use diff -u) if you think they should end up in the mainline, and We'll review and include them.
